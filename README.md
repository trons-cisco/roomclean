This macro wasdeveloped as a Proof of Concept for a customer inquiring about showing a visual image with cleaning instructions post meeting.  

The macro is built to monitor for meeting "disconnect" events, once triggered it will run the "webview display" command to present a hosted image for 2 minutes, at the end of the 2 minute timer, it will run "webview clear".
- The macro works for Webex and SIP calls while in the RoomOS experience
- The Macro works for Microsoft Teams, Webex and SIP calls while in the Microsoft Teams Rooms experience on Cisco Devices
